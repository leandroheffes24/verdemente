Falta configurar Supabase

El panel guarda los productos en una base de datos en la nube (Supabase). Para activarlo, seguí estos pasos una sola vez:

  1. Entrá a app.supabase.com y creá un proyecto (es gratis).
  2. En Project Settings → API, copiá la Project URL y la anon public key.
  3. Pegalas en el archivo supabase-config.js.
  4. En el SQL Editor, corré el script que crea la tabla productos y sus políticas (está en SETUP-SUPABASE.md).
  5. En Authentication → Users, creá el usuario de la dueña (email + contraseña, con "Auto Confirm").

Tenés el paso a paso completo en SETUP-SUPABASE.md.
